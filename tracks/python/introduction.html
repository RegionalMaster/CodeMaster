What is Python?

Python is a high-level, interpreted programming language created by Guido van Rossum and released in 1991. It emphasizes code readability with its notable use of significant whitespace.

Why Choose Python?

  • Easy to Learn: Simple, readable syntax perfect for beginners
  • Versatile: Used in web development, data science, AI, automation, and more
  • Large Community: Extensive libraries and active community support
  • High Demand: One of the most sought-after programming skills
  • Cross-Platform: Runs on Windows, Mac, Linux, and more

Your First Python Program

Let's look at the traditional "Hello, World!" program in Python:

print("Hello, World!")

Expected Output:

Hello, World!

Understanding the Code:

  • print() is a built-in Python function that outputs text to the screen
  • Text in quotes ("") is called a string
  • Python executes code line by line, from top to bottom

Common Beginner Questions

Do I need to install Python?

Yes, to run Python locally. We'll cover installation in the next lesson. For now, you can use online editors.

What version of Python should I use?

We recommend Python 3.x (latest stable version). Python 2 is no longer supported.

What code editor should I use?

Popular choices include VS Code, PyCharm, and IDLE (comes with Python). We'll discuss editors in the setup guide.
