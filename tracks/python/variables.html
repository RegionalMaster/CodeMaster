Understanding Variables

Variables are containers for storing data values. In Python, you don't need to declare a variable's type - Python does it automatically.

# Creating variables
name = "John"
age = 25
height = 1.75
is_student = True

Variable Naming Rules

  • Must start with a letter or underscore
  • Can contain letters, numbers, and underscores
  • Case-sensitive (name and Name are different)
  • Cannot use Python keywords (like if, for, while)

Basic Data Types

Common Data Types

  • str - Text (strings)
  • int - Whole numbers
  • float - Decimal numbers
  • bool - True/False values
# Examples of different data types
text = "Hello"           # str
number = 42             # int
price = 9.99           # float
is_active = True       # bool

# Check the type of a variable
print(type(text))      # 
print(type(number))    # 
print(type(price))     # 
print(type(is_active)) #

Type Conversion

You can convert between different data types using Python's built-in functions:

# Converting between types
age_str = "25"
age_num = int(age_str)    # Convert string to integer

price = 9.99
price_int = int(price)    # Convert float to integer (removes decimal)

number = 42
number_str = str(number)  # Convert number to string

print(age_num + 5)        # 30
print(price_int)         # 9
print(number_str + "!")   # "42!"

Common Type Conversion Functions

  • str() - Convert to string
  • int() - Convert to integer
  • float() - Convert to float
  • bool() - Convert to boolean

Practice Exercises

1. Variable Creation

Create variables to store your personal information:

# Store your information
full_name = "Your Name"
age = 25
height = 1.75
is_programmer = True

# Print the information
print("Name:", full_name)
print("Age:", age)
print("Height:", height)
print("Is Programmer:", is_programmer)

2. Type Conversion

Practice converting between different data types:

# Given values
number_string = "100"
decimal_number = 45.67
whole_number = 23

# Convert and print
result1 = int(number_string) + whole_number
result2 = str(decimal_number) + " dollars"
result3 = float(whole_number)

print(result1)  # Should print: 123
print(result2)  # Should print: 45.67 dollars
print(result3)  # Should print: 23.0

Common Questions

What happens if I convert an invalid string to a number?

Python will raise a ValueError. For example, int("hello") will cause an error.

Can I use spaces in variable names?

No, use underscores instead: first_name instead of first name.

What's the difference between int and float?

int stores whole numbers (42), while float stores decimal numbers (42.0).
