Installing Python

Before you can start programming in Python, you need to install it on your computer. Follow these steps:

System Requirements

  • Windows: Windows 7 or later
  • macOS: macOS 10.11 or later
  • Linux: Most modern distributions
  • Disk Space: At least 100MB free

Step-by-Step Installation

  1. Download Python

    Visit the official Python website and download the latest version for your operating system.

    Important!

    Make sure to check "Add Python to PATH" during installation on Windows!

  2. Run the Installer

    Double-click the downloaded file and follow the installation wizard.

  3. Verify Installation

    Open a terminal or command prompt and type:

    python --version

    Expected Output:

    Python 3.12.0

Choosing an IDE

An Integrated Development Environment (IDE) makes coding easier. Here are our recommended options:

Visual Studio Code (Recommended)

  • Free and open-source
  • Lightweight yet powerful
  • Excellent Python support with extensions
  • Great for beginners and professionals
  • Download VS Code

PyCharm

  • Professional Python IDE
  • Community Edition is free
  • More features but steeper learning curve
  • Download PyCharm

IDLE (Comes with Python)

  • Simple and lightweight
  • Already installed with Python
  • Good for beginners
  • Limited features compared to VS Code/PyCharm

Setting Up VS Code for Python

  1. Install VS Code Extensions

    Open VS Code and install these essential extensions:

    • Python (by Microsoft)
    • Python Extension Pack
    • Code Runner (optional)

  2. Configure Python Interpreter

    Press Ctrl+Shift+P (Windows/Linux) or Cmd+Shift+P (macOS) and type "Python: Select Interpreter"

Common Setup Issues

'python' is not recognized (Windows)

Solution: Reinstall Python and make sure to check "Add Python to PATH" during installation.

Multiple Python versions installed

Solution: Use the Python Launcher (py command) or specify the full path to the desired Python version.

Permission errors during installation

Solution: Run the installer as administrator (Windows) or use sudo (Linux/macOS).
