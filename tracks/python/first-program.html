Creating Your First Python File

Let's start by creating a simple Python program. Follow these steps:

  • Create a New File

    Open your chosen IDE (VS Code, PyCharm, or IDLE) and create a new file called hello.py

  • Write Your Code

    Type the following code:

    # This is my first Python program
print("Hello, Python!")
print("I'm learning to code!")

  • Save the File

    Save the file with the .py extension (e.g., hello.py)

  • Run the Program

    Open a terminal and navigate to your file's location, then type:

    python hello.py

    • Expected Output:

    Hello, Python!
I'm learning to code!

    Understanding the Code

    Code Breakdown

    • # This is my first Python program - A comment (Python ignores lines starting with #)
    • print() - A built-in function that displays text on the screen
    • "Hello, Python!" - A string (text enclosed in quotes)

    Common Beginner Mistakes

    Watch out for:

    • Missing quotes around text
    • Using single quotes (') and double quotes (") inconsistently
    • Forgetting parentheses () after print
    • Incorrect file extension (should be .py)

    Practice Exercises

    Try these variations to get comfortable with Python:

    1. Personal Greeting

    Create a program that prints your name and age:

    # My personal greeting
print("My name is Alice")
print("I am 25 years old")

    Expected Output:

    My name is Alice
I am 25 years old

    2. ASCII Art

    Create a simple picture using text:

    # Drawing with text
print("  *  ")
print(" *** ")
print("*****")

    Expected Output:

      *  
 *** 
*****

    Common Questions

    Why isn't my program running?

    Make sure you're in the correct directory and the file has a .py extension.

    What's the difference between print('text') and print("text")?

    In Python, you can use either single (') or double (") quotes for strings. They work the same way.

    Do I need to include comments (#)?

    Comments are optional but recommended. They help explain your code to others (and yourself later).
