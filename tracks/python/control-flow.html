Python Control Flow

Control flow is the order in which individual statements, instructions, or function calls are executed in a program. In this tutorial, you'll learn how to control the flow of your Python programs using conditional statements and loops.

If Statements

Python uses if statements for conditional execution. Here's an example:

Basic if statement example 
# Try changing the number to see different results
number = 7

if number > 10:
    print("Number is greater than 10")
elif number > 5:
    print("Number is greater than 5 but less than or equal to 10")
else:
    print("Number is less than or equal to 5")
Basic if statement 
age = 18
if age >= 18:
    print("You are an adult")
else:
    print("You are a minor")
if-elif-else chain 
score = 85
if score >= 90:
    grade = "A"
elif score >= 80:
    grade = "B"
elif score >= 70:
    grade = "C"
else:
    grade = "F"

For Loops

For loops are used to iterate over sequences. Here's an example:

Basic for loop example 
# A simple for loop example
fruits = ["apple", "banana", "orange"]

for fruit in fruits:
    print(f"I like {fruit}s!")
Basic for loop 
fruits = ["apple", "banana", "orange"]
for fruit in fruits:
    print(fruit)
range() function 
# Print numbers 0 to 4
for i in range(5):
    print(i)

# Print numbers 2 to 5
for i in range(2, 6):
    print(i)

# Count by 2s
for i in range(0, 10, 2):
    print(i)

While Loops

While loops repeat a block of code as long as a condition is true.

Basic while loop 
count = 0
while count < 5:
    print(count)
    count += 1

⚠️ Important Note

Be careful with while loops! Always make sure the condition will eventually become false, or you'll create an infinite loop.

Break and Continue

Break and continue statements give you more control over your loops.

Break statement 
for i in range(10):
    if i == 5:
        break  # Exit the loop
    print(i)
Continue statement 
for i in range(5):
    if i == 2:
        continue  # Skip this iteration
    print(i)

Practice Exercises

  1. Write a program that prints numbers from 1 to 100. For multiples of 3, print "Fizz" instead of the number, and for multiples of 5, print "Buzz".
  2. Create a simple number guessing game using a while loop.
  3. Write a program that finds all even numbers between 1 and 50 using a for loop and continue statement.
