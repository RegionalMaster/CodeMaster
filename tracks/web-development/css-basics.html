CSS Basics

CSS (Cascading Style Sheets) is used to style and layout web pages. It determines how HTML elements should be displayed on screen, paper, or other media.

CSS Syntax

CSS consists of selectors and declarations:

Basic CSS syntax 
selector {
    property: value;
    another-property: value;
}
Example with multiple selectors 
h1 {
    color: blue;
    font-size: 24px;
}

p {
    color: gray;
    line-height: 1.6;
}

CSS Selectors

CSS selectors define which elements to style:

CSS Selector Examples 
/* Element selector */
p {
    color: blue;
}

/* Class selector */
.highlight {
    background-color: yellow;
}

/* ID selector */
#header {
    font-size: 24px;
}

/* Descendant selector */
div p {
    margin: 10px;
}

/* Multiple selectors */
h1, h2, h3 {
    font-family: Arial, sans-serif;
}

Colors and Backgrounds

CSS provides various ways to specify colors and backgrounds:

Colors and backgrounds 
/* Color names */
.text-red {
    color: red;
}

/* Hex colors */
.text-blue {
    color: #0000FF;
}

/* RGB and RGBA */
.text-green {
    color: rgb(0, 255, 0);
}
.transparent-bg {
    background-color: rgba(0, 0, 0, 0.5);
}

/* Backgrounds */
.hero {
    background-image: url('image.jpg');
    background-size: cover;
    background-position: center;
}

Box Model

The CSS box model is fundamental to layout:

CSS Box Model Example 
.box {
    /* Content dimensions */
    width: 200px;
    height: 100px;

    /* Padding */
    padding: 20px;
    /* Or individual sides */
    padding-top: 10px;
    padding-right: 20px;
    padding-bottom: 10px;
    padding-left: 20px;

    /* Border */
    border: 2px solid black;
    border-radius: 5px;

    /* Margin */
    margin: 10px;
    /* Or individual sides */
    margin-top: 10px;
    margin-right: 20px;
    margin-bottom: 10px;
    margin-left: 20px;
}

Flexbox Layout

Flexbox makes it easy to create flexible layouts:

Flexbox Example 
/* Container */
.flex-container {
    display: flex;
    justify-content: space-between;
    align-items: center;
    flex-wrap: wrap;
    gap: 20px;
}

/* Flex items */
.flex-item {
    flex: 1;
    min-width: 200px;
    padding: 20px;
}

/* Responsive flex items */
@media (max-width: 768px) {
    .flex-item {
        flex: 100%;
    }
}

Responsive Design

Make your website responsive with media queries:

Responsive Design Example 
/* Base styles */
.container {
    width: 90%;
    max-width: 1200px;
    margin: 0 auto;
}

/* Tablet styles */
@media screen and (max-width: 768px) {
    .container {
        width: 95%;
    }
    
    .nav-menu {
        flex-direction: column;
    }
}

/* Mobile styles */
@media screen and (max-width: 480px) {
    body {
        font-size: 14px;
    }
    
    .hide-on-mobile {
        display: none;
    }
}

Layout and Positioning

CSS offers various ways to control element positioning:

Position and display 
/* Position */
.relative {
    position: relative;
    top: 10px;
    left: 20px;
}

.absolute {
    position: absolute;
    top: 0;
    right: 0;
}

/* Display */
.block {
    display: block;
}

.inline {
    display: inline;
}

.flex-container {
    display: flex;
    justify-content: center;
    align-items: center;
}

Practice Exercise

Create a simple card design that includes:

  1. A header with a background color
  2. An image that scales properly
  3. Some text content with proper spacing
  4. A border and rounded corners
  5. Hover effects for interactivity
Card example 
.card {
    width: 300px;
    border: 1px solid #ddd;
    border-radius: 8px;
    overflow: hidden;
    transition: transform 0.3s ease;
}

.card:hover {
    transform: translateY(-5px);
    box-shadow: 0 4px 8px rgba(0,0,0,0.1);
}

.card-header {
    background-color: #f5f5f5;
    padding: 15px;
}

.card img {
    width: 100%;
    height: auto;
}

.card-content {
    padding: 15px;
}
